- Vagrant 下载离线box:

  ```
  https://app.vagrantup.com/generic/boxes/centos7/versions/3.0.30/providers/virtualbox.box

  centos/7: 
  https://app.vagrantup.com/centos/boxes/7/versions/2004.01/providers/virtualbox.box
  ```
